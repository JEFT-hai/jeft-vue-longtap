# jeft-vue-longtap

## Installation

``` js
npm install jeft-vue-longtap --save
```

## Usage

``` js
/**
 * @description: 长按事件
 * @param {arg} 触发事件的时间
 * @return: e touchstart事件的event el触发的dom
 * 支持PC与移动端
 * eg: v-longTap:600='longTap'
 */
```
